=============================================
Neuron v4.7 — Production Portability Release
=============================================

Welcome to Neuron! 

Neuron is an open-source, offline-first semantic search engine designed to help you find your files by meaning, not just keywords. It includes its own bundled AI embedding matrix natively packaged for Windows.

---------------------------------------------
⚠️ Note regarding Windows SmartScreen ⚠️
---------------------------------------------
Because Neuron is newly released and an indie open-source project, it does not currently use an expensive Extended Validation (EV) Code Signing Certificate. 

If Windows presents a blue "Windows protected your PC" SmartScreen warning:
1. Click "More info" 
2. Click "Run anyway"

Neuron is completely safe, does not use telemetry, and all source code is publicly accessible on our GitHub repository. This warning is a standard Microsoft precaution for newly published executables without expensive corporate signing keys and will naturally resolve automatically as Neuron gains more downloads globally.

---------------------------------------------
Requirements
---------------------------------------------
* Windows 10/11 (64-bit)
* 4 GB RAM minimum (8 GB highly recommended)
* 500 MB free disk space

You do not need to install Python or any dependencies on your system.
This installer brings a completely self-contained deployment to your machine.
