# Neuron – Refactor to 9/10 Checklist

Ordered, low-risk, file-by-file plan to improve code quality and maintainability
without changing runtime behaviour. Items are roughly ordered by impact and safety.

---

## A — Architecture cleanup (desktop-first)

- [x] **Deprecate `run.py`** — add top-of-file notice marking it as the legacy
  web-mode entry point (FastAPI + pywebview). Desktop users should use
  `run_desktop.py`. File kept for reference; do not remove until a full
  breaking-change release.

- [x] **Deprecate `app/main.py`** — add top-of-file notice marking it as the
  FastAPI application that is only needed when running in web/server mode via
  `run.py`. The desktop path (`run_desktop.py → services/desktop_service.py`)
  never touches this file.

- [x] **Deprecate `Dockerfile` / `docker-compose.yml`** — add header comments
  noting these are web-mode (server) deployment files only, not used by the
  desktop app.

- [x] **Remove `services_desktop.py` (root)** — dead file that references
  non-existent import paths (`core.embeddings.model`, `core.indexing.faiss_store`,
  `core.ingestion.file_reader`, `core.search.ranker`). It is not imported by any
  other module. Removing it eliminates a source of confusion.

- [ ] *Deferred* — Move all web-only files (`run.py`, `api/`, `app/main.py`,
  `Dockerfile`, `docker-compose.yml`) under a `web/` sub-package. Deferred
  because it requires updating `import` paths inside those files and is
  higher-risk without a full test suite for the web path.

---

## B — Module structure and code hygiene

- [x] **`app/config.py`** — add inline comments marking `API_HOST`, `API_PORT`,
  `API_RELOAD` as web-mode only; mark `METADATA_PATH` and `INDEXED_PATHS_DB` as
  deprecated (replaced by `SQLITE_DB_PATH`).

- [x] **`core/__init__.py` / sub-package `__init__.py`** — add focused `__all__`
  lists to `core/embeddings/__init__.py`, `core/indexing/__init__.py`,
  `core/search/__init__.py`, `core/ingestion/__init__.py`, `core/watcher/__init__.py`
  so that the public surface of each sub-package is explicit.

- [ ] *Deferred* — Break up `run_desktop.py` (≈1200 lines) into dedicated modules
  (`ui/hotkey.py`, `ui/result_card.py`, `ui/settings_panel.py`, etc.). Deferred
  because it requires careful Qt parent/signal wiring and is higher-risk.

- [ ] *Deferred* — Extract `ICON_MAP` constant from `run_desktop.py` into
  `ui/file_icons.py` alongside `get_file_icon()`. Low risk but coupled to the
  larger split above.

- [ ] *Deferred* — Replace repeated `platform.system() == "Windows"` guards in
  `run_desktop.py` and `services/desktop_service.py` with a small `_IS_WIN`
  constant. Trivial but out of scope for this PR.
